B-Spline drawer

Start by clicking on the canvas to add points.
You can remove points by right-clicking on them.
After clicking on each existing point it will be marked by being red.
You can change the weight of selected point by changing the W slider. The slider is only visible when a point is selected
The point with U value is being shown on the curve as a green point.
The geometry shape of the deBoor algorithm has been drawn by yellow lines
